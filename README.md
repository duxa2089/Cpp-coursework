# Программа для управления деятельностью склада торгового предприятия
Программа должна обеспечивать возможность выполнения перечисленных ниже функций:

1. Функция записи товара в коллекцию.
2. Функция редактирования информации о товаре.
3. Функция удаления информации о товаре.
4. Функция сортировки записи в алфавитном порядке.
